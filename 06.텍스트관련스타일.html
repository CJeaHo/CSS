텍스트관련 스타일

color : 텍스트의 색상 지정

	선택자 {
		color: 색상명|16진수|rgb(r, g, b)|rgba(r, g, b, a)|hsl(h, s, l)|hsla(h, s, l, a)
		a:투명도(0~1), h:색상값(0~360), s:채도(0~100%), l:밝기(0~100%)
	}

색상 지정하기

text-decoration : 텍스트에 줄을 긋거나 없앨 때 사용

	text-decoration: none|underline|overline|line-through
	none: 줄 삭제
	underline: 밑줄
	overline: 윗줄
	line-through: 가운데줄
  • 텍스트 영역 아래 줄 긋기
  • 텍스트 영역 중간에 줄 긋기
  • 텍스트 영역 위에 줄 긋기
네이버로 이동

text-transform : 영문텍스트의 대소문자 변환시 사용

	선택자 {text-transform: uppercase|lowercase|capitalize}
	uppercase: 소문자를 대문자로
	lowercase: 대문자를 소문자로
	capitalize: 첫글자만 대문자로
  1. uppercase: 모든 영문자를 대문자로
  2. LOWERCASE: 모든 영문자를 소문자로
  3. capitalize: 영문자의 첫글자만 대문자로

text-shadow : 텍스트에 그림자 효과를 줄 때 사용

		선택자 {text-shadow: 가로거리(px) 세로거리(px) [번짐정도px][그림자 색상]}
